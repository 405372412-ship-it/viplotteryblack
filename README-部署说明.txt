VIP礼品直播抽奖软件 - 单项目组静态部署包

文件结构：
admin.html  后台管理页
screen.html 前台抽奖页
assets/     图片和音乐素材

访问地址示例：
https://你的域名/group-a/admin.html
https://你的域名/group-a/screen.html

使用方式：
1. 上传本文件夹内所有文件到静态网站目录。
2. 工作人员打开 admin.html，输入后台密码后配置活动。
3. 后台点击“打开前台抽奖页”，或让大屏直接打开 screen.html。
4. 前台进入轮次后执行开始抽奖、抽奖结束。
